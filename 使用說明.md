# IHLCA 計算工具（Integrated Hybrid LCA） — 使用說明

> 本版本為標準版的延伸，新增三項功能：NAS 共用資料夾載入、資料品質 Pedigree 評分、DB 上傳格式 Excel 輸出。
>
> **本工具只計算 Integrated Hybrid LCA**（必須提供 T 製程資料）。如需純 IO LCA，請使用組織內既有的 800 筆 master Excel；本工具的用途是「將既有 IO LCA 個別係數**升級**為 Integrated」。

## 強化版新增功能

### ① 內建 A 矩陣 + 可選自訂 B_A 上傳（側欄）

工具啟動時會**自動載入內建 A 矩陣**（台灣主計總處 110 年產業關聯表，163 部門）與**預設 B_A 碳強度向量**，組員不需操作即可開始計算。

側欄提供「上傳自訂 B_A」功能（**選用**）：

- 下載 `B_A填報範本.xlsx`（單一分頁、含部門名稱對照）
- 填妥後上傳即可覆寫預設 B_A
- 適用情境：能源係數公告更新、特殊版本 B_A 需暫時套用

### ② 資料品質 Pedigree 評分（輸入分頁，每張材料表下方）

每張原物料／能資源表的下方都有「🔍 編輯資料品質評分」摺疊區，可對該表所有材料逐項輸入：

- **活動數據** 5 維度：Re（可靠性）、Co（完整性）、Ti（時間相關性）、Ge（地理相關性）、Te（技術相關性）
- **排放係數** 5 維度：同上

分數 1（最佳）至 5（最差），預設值為 3。

### ③ 產生 DB 上傳格式 Excel（計算結果分頁底部）

計算完成後，「📊 計算結果」分頁底部會出現「🌐 產生 DB 上傳格式 Excel」區塊。填妥後設資料（PCCES 編碼、版次、計算方法等）後按「📦 產生 DB 上傳 Excel」，會輸出 6 分頁 Excel：

1. `products-import-template` — 26 欄符合網站資料庫上傳格式
2. `原物料投入 T` — 製程配比
3. `原物料單價Cu` — 單價清單
4. `原物料排放係數B` — 含 163 部門編號／名稱
5. `數據品質` — Pedigree 5×2 評分表
6. `碳足跡熱點` — 排序後的熱點清單

檔名自動命名為 `{PCCES編碼}-{方法}-{版次}.xlsx`，例：`M1658121006-IO-V1.0.xlsx`。

## 第一次使用（只需做一次）

## 第一次使用（只需做一次）

1. 解壓縮收到的 zip 檔案
2. 對「啟動碳足跡計算工具.command」按右鍵 → 選「開啟」
   （第一次需要右鍵開啟，之後就可以直接雙擊）
3. 跳出安全性警告時，點「開啟」確認
4. 等待套件安裝完成（約 1-2 分鐘），瀏覽器會自動開啟

## 之後每次使用

直接雙擊「啟動碳足跡計算工具.command」，瀏覽器會自動開啟。

## 注意事項

- 使用期間請勿關閉自動出現的終端機視窗
- 關閉工具：直接關閉終端機視窗即可
- 若瀏覽器沒有自動開啟，請手動輸入：http://localhost:8501

## 系統需求

- macOS
- Python 3（若未安裝請至 https://www.python.org 下載）
- 網路連線（僅第一次安裝套件時需要）
